## ターゲット求人の定義

- **職種**：自社の情報システム部門に所属し、社内ネットワーク（LAN/WAN/VPN等）の
  設計・構築・運用・保守を担うポジション
- **雇用形態**：正社員のみ
- **除外条件**：勤務場所が自社オフィス・本社以外になるポジション
  （客先常駐・出向常駐・常駐派遣等）
- **企業条件**：東証上場企業、年間休日120日以上
- **勤務場所**：都道府県・市区町村単位での絞り込みを行いたい

## 調査事項

### 1. 対象企業・情報ソース
- 東証プライム上場企業のうち、社内ネットワークエンジニアを直接雇用しやすい業界の傾向
  （金融・製造・流通・メディア等、情シス規模が大きい業界）
- 採用情報の掲載チャネル（公式採用ページ / Green / Wantedly / LinkedIn 等）
- 公式採用ページのURL構造・更新頻度の傾向

### 2. 情報取得の技術的手段
- 公開APIが存在するか（Indeed / 求人ボックス / Wantedly / LinkedIn 等）
- RSS / JSONフィードを提供している採用サイトの有無

### 3. フィルタリング条件の実現可能性
以下を自動判定・フィルタできるか評価してほしい：
- 正社員雇用
- 年間休日120日以上
- 上場企業（東証）
- 職種：社内ネットワークエンジニア（情シス所属・自社勤務）
- 勤務場所が客先常駐・出向でないこと（自社オフィス勤務のみ）
- 勤務場所：都道府県・市区町村単位での絞り込み

### 4. 法的・利用規約上の注意点
- 主要採用ページのrobots.txtによるスクレイピング制限の傾向
