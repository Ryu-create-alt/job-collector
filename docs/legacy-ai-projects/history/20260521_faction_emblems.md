## 依頼内容

アズールレーン各陣営のエンブレムを SVG で実装したい。
**本家アズールレーンの公式エンブレムに匹敵するレベルの繊細さ・精密さ**を目標とする。

### 技術仕様

- **viewBox**: `0 0 512 512`（全エンブレム共通）
- **制約**: React Native SVG (`react-native-svg`) で使用するため、以下のみ使用可能
  - `<Path d="..." />`（メイン）、`<Circle cx r />`、`<G transform="rotate/translate/scale" />`
  - **使用不可**: `<filter>`, `<defs>`, `<linearGradient>`, `<clipPath>`, テキスト要素
- **色**: 単色。全 fill は実装時に差し替えるため、出力パスの fill 値は不問（あとで置換）
- **品質目標**: アズールレーン公式サイト・ゲーム内の陣営紋章レベルの繊細さ。単純な幾何学形状ではなく、複雑なベジェ曲線で有機的・紋章的な美しさを表現すること

**2. ロイヤル (Royal Navy) — 王冠付き紋章ライオン（rampant lion）**

**3. 重桜 (Sakura Empire) — 桜花紋（日本家紋スタイル）**

- おしべ・葉脈のディテール、旭日旗・海軍軍旗水準のシンボル

**4. 鉄血 (Iron Blood) — アイアンクロス + ゲルマン鷲**

**5. アイリス (Iris Libre) — フルール・ド・リス（百合紋）**

**6. ヴィシア (Vichya Dominion) — 交差剣**

### 品質チェックリスト
- 各パスのベジェ曲線は制御点を多用（単純な円弧・直線を避ける）
- 実際の紋章・家紋を参考にした本格的なプロポーション
- 512x512 を適切に使用（余白は各辺 20〜30px 程度）
- 単色で表示した際に識別性が高い（シルエットとして機能する）
