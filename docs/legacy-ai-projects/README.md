# 東証上場企業 社内ネットワークエンジニア求人直接収集ツール (プロトタイプ)

大手上場企業のキャリア採用ページから「社内ネットワーク・正社員・自社勤務限定」求人を直接クロールし、自動判定・フィルタしたうえでWebダッシュボードに一覧表示する個人用ツールです。

## 技術スタック
- **言語**: Python 3.10+
- **ブラウザ自動操作**: Playwright
- **パーサー**: BeautifulSoup4
- **データベース**: SQLite
- **ダッシュボード**: Streamlit

## セットアップ手順

1. **依存ライブラリのインストール**
   ```bash
   pip install -r requirements.txt
   ```

2. **Playwright用ブラウザのインストール**
   ```bash
   python -m playwright install chromium
   ```

## 使い方

1. **求人の収集実行**
   以下のコマンドを実行すると、`config/companies.json` に定義された企業ページを巡回し、求人をスクレイピング・フィルタリングして `jobs.db` に保存します。
   ```bash
   python run.py
   ```

2. **Webダッシュボードの表示**
   収集した求人をブラウザ上で閲覧・検索・フィルタリングするための高級ダッシュボードを起動します。
   ```bash
   streamlit run src/dashboard.py
   ```

## フィルター条件の仕組み
- **雇用形態判定**: 求人内の「雇用形態」または構造化データ (JSON-LD) から「正社員」を自動検出。
- **年間休日判定**: `(年間休日|年休)\s*[:：]?\s*(\d{3})\s*日` の正規表現で120日以上か判定（取得不可の場合は「完全週休2日」＋「祝日」等の記述で推定）。
- **客先常駐の排除**: 「客先常駐」「SES」「派遣」「クライアント先」などのキーワードが含まれる求人を自動で除外。
- **職種判定**: 求人タイトルや本文中の技術キーワード（LAN/WAN/VPN/Cisco等）および「社内」「インハウス」といったキーワードのスコアリングによる判定。
